# Personalized Shopping Agent – User Profile
#
# This user is a **research-heavy, value-conscious shopper** who:
#
# - Spends significant time comparing products before committing
# - Trusts ratings and review volume over brand prestige in low-cost categories
# - Becomes very cautious (risk-averse) in high-stakes categories like electronics
# - Prefers items with clear return/refund policies
# - Dislikes suspiciously cheap products with few reviews
# - Values mid-range options that balance quality and price
# - Tends to shortlist 2-3 candidates before adding one to cart
# - Rarely impulse-buys; usually revisits items
# - Will pay a moderate premium for reliability but not for luxury branding alone
# - Cares about seller reputation (avoids unknown sellers)
#
# ## Decision Process
# 1. Search broadly first
# 2. View several options
# 3. Compare top candidates in detail
# 4. Shortlist 2-3 items
# 5. Add best value to cart
# 6. Buy only after reviewing cart contents
#
# ## Regret Patterns
# - Returns items that looked good online but had poor build quality
# - Feels satisfied when an item matches the review consensus
# - Regrets skipping a good deal (but regrets impulse buys more)
